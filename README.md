The program searches for the shortest games for boards of size 4x4 - 12x12 for two starting positions: standard <img src="https://slovesnov.rf.gd/img/reversi/js1.png"> and nonstandard <img src="https://slovesnov.rf.gd/img/reversi/js3.png"> and for three types of game ends: only black chips, only white chips, white and black chips. Below the table of found games is shown, after the sequence of moves their number is given, as well as the total number of found positions of minimum game length of the given type.

Программа делает поиск кратчайших игр для досок размера 4x4 - 12x12 для двух стартовых позиций: стандартной <img src="https://slovesnov.rf.gd/img/reversi/js1.png"> и нестандартной <img src="https://slovesnov.rf.gd/img/reversi/js3.png"> и для трех типов концов игры: только черные фишки, только белые фишки, белые и черные фишки. Внизу предоставлена таблица найденных игр, после последовательности ходов указано их количество, а также общее количество найденных позиций минимальной длины игры заданного типа.
